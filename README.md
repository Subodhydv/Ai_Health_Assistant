# 🩺 AI Health Assistant

An AI-powered web app that analyzes user symptoms and predicts possible conditions with risk levels.

## 🚀 Features
- Symptom-based disease prediction
- Risk classification (Low / Medium / High)
- AI-based advice with fallback system
- Clean UI using Streamlit

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API

## ⚙️ How to Run

```bash
pip install streamlit openai
streamlit run app.py
